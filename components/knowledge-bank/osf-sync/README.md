# OSF Sync

Responsibilities:
- read OSF projects, components, files, and preprints
- normalize OSF metadata into DB tables
- link OSF records to paper and concept notes

Minimum synced fields:
- node id
- node title
- node category
- file id and file path
- file provider and version
- preprint id and DOI
- license and publication status
- contributor ids/roles

Policy:
- respect private and embargoed visibility settings
- treat publish/update actions as explicit human-approved workflows
