# Obsidian Vault

This folder is your Obsidian free-tier vault root.

Guidelines:
1. Keep all notes in markdown.
2. Use wiki links (`[[concept-name]]`) for graph relationships.
3. Use frontmatter keys for provenance and status:
- `source`
- `doi`
- `date_added`
- `confidence`
- `evidence_level`

Obsidian is the exploration layer. Canonical metadata remains in DB + git versioned files.
